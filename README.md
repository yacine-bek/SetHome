# HomeSet
HomeSet is a lightweight Minecraft plugin that gives you essential home commands.

# What This Plugin Does
- Set a personal home location that only you can access.
- Easily teleport back to your home whenever you want.
- No complicated setup - just install and use!
# How to Use
- Type /sethome to save your current location as home.
- Type /home to instantly teleport back to that spot.
Perfect for survival servers, keeping your base within reach no matter how far you explore!
